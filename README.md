# LLM QA Tester
A professional testing toolkit for AI chatbots, LLMs, and RAG systems.

## Features
- Batch question-answering evaluation
- Hallucination detection & automated scoring
- Prompt injection & system prompt leak testing
- RAG factual accuracy validation
- CSV report generation
